# ⚡ Quick Start - WordPress Divination Blocks

Get your plugin installed and selling in **30 minutes**!

## Step 1: Test Locally (10 min)

### Install WordPress Locally

**Option A: Local by Flywheel** (Easiest)
1. Download [Local by Flywheel](https://localwp.com/) (free)
2. Install and create new site
3. Site name: "Divination Test"
4. Start site

**Option B: XAMPP/MAMP**
1. Install XAMPP or MAMP
2. Download WordPress from wordpress.org
3. Set up database and install

**Option C: Docker**
```bash
docker run -p 8080:80 -d --name wp-test wordpress
```

### Install Plugin

1. Copy plugin folder to WordPress:
   ```bash
   cp -r plugins/wordpress-divination-blocks /path/to/wordpress/wp-content/plugins/
   ```

2. Go to WordPress admin → Plugins
3. Activate "Divination Blocks"

### Test It

1. Create new page
2. Add block → Search "Oracle Card"
3. Insert block
4. Publish and view!

## Step 2: Build for Distribution (5 min)

### Create ZIP file

```bash
cd plugins
zip -r divination-blocks-v1.0.0.zip wordpress-divination-blocks/ \
  -x "*.git*" -x "*.DS_Store" -x "*node_modules*"
```

This creates `divination-blocks-v1.0.0.zip` ready for upload!

## Step 3: Choose Sales Channel (15 min)

### Option A: CodeCanyon (Recommended First)

**Why**: Instant audience, built-in trust, easy setup

1. **Create Account**
   - Go to [CodeCanyon](https://codecanyon.net/)
   - Sign up as Author
   - Complete tax info

2. **Submit Plugin**
   - Upload → "WordPress" category
   - Upload ZIP file
   - Title: "Divination Blocks - Oracle Cards & Spiritual Gutenberg Blocks"
   - Price: $49 (you get $24.50 after 50% commission)
   - Description: (see template below)
   - Screenshots: 6-8 images showing blocks in action
   - Demo: Optional but helpful

3. **Wait for Review**
   - Usually 3-7 days
   - They'll request changes if needed
   - Once approved → Live!

4. **First Sale**
   - Usually within first week
   - Promote in WP communities

### Option B: Your Own Site

**Why**: 100% revenue, own customer data

1. **Set Up Sales Page**
   - Use WooCommerce
   - Or Gumroad/LemonSqueezy
   - Price: $79-149 (higher than CodeCanyon)

2. **Licensing**
   - Use [Software License Manager](https://wordpress.org/plugins/software-license-manager/)
   - Or [Easy Digital Downloads](https://easydigitaldownloads.com/) with licensing

3. **Payment Processing**
   - Stripe (2.9% + 30¢)
   - PayPal (similar fees)

### Option C: WordPress.org (Freemium)

**Why**: Massive reach, build trust, recurring revenue

1. **Prepare Free Version**
   - Remove Pro features
   - Keep Oracle Card block only
   - Add "Upgrade to Pro" notices

2. **Submit to WP.org**
   - Go to [WordPress Plugin Directory](https://wordpress.org/plugins/developers/)
   - Submit plugin
   - Follow their guidelines
   - Review takes 1-2 weeks

3. **Upsell to Pro**
   - Link to your site for Pro version
   - 5-10% conversion rate typical
