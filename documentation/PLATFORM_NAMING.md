## Naming Best Practices

### DO:
✅ Keep names under 3 words  
✅ Make it memorable and pronounceable  
✅ Include primary keyword when possible  
✅ Consider SEO from day one  
✅ Test name with target audience  
✅ Check domain/social availability  
✅ Think about verbal communication (phone, word-of-mouth)  
✅ Consider international pronunciation  

### DON'T:
❌ Use overly complex/confusing names  
❌ Rely on acronyms (hard to remember)  
❌ Choose names too similar to competitors  
❌ Use trendy slang that will age poorly  
❌ Ignore trademark issues  
❌ Make it too generic (gets lost in search)  
❌ Use numbers or special characters  

---

## SEO Naming Tips

1. **Include Target Keyword**: "Resume Scanner" vs just "Ronna"
2. **Make it Descriptive**: Users should know what you do from the name
3. **Brand + Function**: "BlackLabb Oracle Cards" combines both
4. **Local SEO**: Add location for local services (not applicable here)
5. **Long-tail Keywords**: "Free Oracle Card Reading" in meta descriptions

## Trademark Considerations

Before finalizing any name:

1. **Search USPTO Database**: Check for existing trademarks
2. **Google the Name**: See what's already using it
3. **Check Social Handles**: Ensure availability
4. **Domain Variations**: Check .com, .io, .app availability
5. **International**: Check if name means something offensive elsewhere

---

## Name Testing Checklist

For each platform name:

- [ ] Easy to spell when heard verbally?
- [ ] Memorable after one mention?
- [ ] Makes sense to target audience?
- [ ] Available as social media handle?
- [ ] Passes "mom test" (your mom would understand it)?
- [ ] No negative connotations?
- [ ] SEO-friendly (includes keywords)?
- [ ] Brandable (can build identity around it)?
- [ ] Scalable (works as you grow)?
- [ ] Legally clear (no trademark conflicts)?
