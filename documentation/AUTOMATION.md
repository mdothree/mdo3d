## Guardrails

### Permitted Actions
- API key management (read/write)
- Data entry and updates
- Content drafting
- Monitoring and reporting
- Publishing to GitHub
- Infrastructure deployment

### Prohibited Actions
- ❌ Payment or financial transactions
- ❌ Account deletion
- ❌ Password handling or modifications
- ❌ Negative content or bad speech
- ❌ Bold claims or misrepresentations

### Requires Approval
- Sending messages
- Posting comments
- Publishing content
- Any action affecting accounts
